# README for WeebUp (MeetUp Clone)

<a href="https://meetup-api-project.herokuapp.com/">Weeb Up</a> is a clone website of Meetup, but specifically for anime enjoyers and enthusiasts. Weebup is used for creating groups and organizing events by the anime loving community, new and old. Check out the website using Demo User to explore the current CRUD features available for groups and events.

# Get Started

## How to set up the repo to run the project locally

1. Clone the repo
2. Run npm install in both /authenticate-me/backend and /authenticate-me/frontend
3. Run npm start in both directories in separate terminals
4. To see the locally hosted site, open your browser to http://localhost:3000/

# Features and functionality

The main feature in this site revolves around groups, more specifically, creating, reading, updating, and deleting groups.

1. To create a group, user must be logged in. There is a demo user available under the login form for easy access to the site's features. Start by loggin in via newly creating account or demo user then navigating to 'Start a group'

   <img src="https://i.imgur.com/UvLRrjg.png" width="600">

2. Follow the steps to create a group.

   <img src="https://i.imgur.com/79peRds.png" width="600">

3. Once you created a group, you can navigate to the group page and read, edit, or delete the group. From the group page, you can also create an event for the group.

   <img src="https://i.imgur.com/C5lCdco.png" width="600">

4. To create an event, click on "Create Event" from the group page that you just created and fill in the form information with valid data.

   <img src="https://i.imgur.com/BUZtOD5.png" width="600">

5. After creating an event, you can navigate to it through the events page and read/delete the contents as well.

   <img src="https://i.imgur.com/cVX6A7P.png" width="600">
